# Simple To-do App
 
Using React, FastAPI, and SQLite

## Description

To-do (pun intended)

## Getting Started

### Dependencies

* React
* FastAPI (pip3 package)
* SQLite
* SQLAlchemy (pip3 package)
* pyjwt (pip3 package)
* passlib[bcrypt] (pip3 package)

### Run it Locally

* Backend (while in directory)
```sh
source venv/bin/activate
uvicorn main:app --reload
```
  
* Frontend:
```sh
npm start
```
